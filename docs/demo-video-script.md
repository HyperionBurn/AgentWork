# AgentWork — Demo Video Script

> **Duration**: 3 minutes
> **Audience**: Hackathon judges (Agentic Economy on Arc, lablab.ai)
> **Tone**: Confident, technical, "show don't tell"

---

## Scene 1: Hook (0:00 — 0:15)

**Narration**: "What if AI agents could pay each other — instantly — for five thousandths of a dollar per task?"

**Screen**:
- Dashboard loads, dark theme, agents online (green dots)
- Camera pans across hero stats: "0 On-Chain Transactions" ready to change
- Pause on "Run Demo" button — big, gradient purple-blue

---

## Scene 2: Architecture (0:15 — 0:45)

**Narration**: "AgentWork is a marketplace where an orchestrator decomposes complex tasks, hires specialist agents via x402 nanopayments on Arc L1, and settles through Circle Gateway — with 60+ verified on-chain transactions."

**Screen**:
- Architecture diagram overlay (PNG or Mermaid):
  ```
  Orchestrator → [Research, Code, Test, Review] Agents
       ↓                  ↓
  Circle Gateway → Arc L1 (USDC native gas)
  ```
- Highlight: "$0.001 per transaction", "<5 second settlement", "USDC native gas"

---

## Scene 3: Live Demo (0:45 — 1:45)

**Narration**: "Let's run it. One click."

**Screen**:
1. Click "🚀 Run Demo" button
2. Button transitions to "⚡ Running (15 runs)..."
3. Transaction waterfall animates:
   - "Research Agent — $0.005 — ✅ confirmed on Arc [arcscan link]"
   - "Code Agent — $0.005 — ✅ confirmed"
   - "Test Agent — $0.005 — ✅ confirmed"
   - "Review Agent — $0.005 — ✅ confirmed"
   - Repeat cascading with waterfall animation
4. Agent responses expand — show real LLM content (or rich mock)
5. Hero stats counter climbing: "47 On-Chain Transactions"

**Key moment**: Multiple arcscan links visible and clicked to verify

---

## Scene 4: Evidence (1:45 — 2:15)

**Narration**: "Don't trust, verify. Every transaction is independently verifiable on Arc."

**Screen**:
1. Click "📊 Evidence" link in header
2. Evidence page loads: "XX Verified On-Chain Transactions" hero counter
3. Scroll through table — tx hashes with arcscan links
4. Click one arcscan link — browser opens arcscan.io showing real transaction
5. Click back — show "Download CSV" button
6. Summary: total amount, agents used, average cost per transaction

---

## Recording Tips

1. **Pre-run the demo once** before recording to warm caches
2. **Close other tabs** — reduce visual noise
3. **Use 1920×1080** — standard resolution for screen recording
4. **Show real arcscan links** — click at least 2-3 to prove they work
5. **If demo fails** — cut to pre-recorded backup, never show errors
6. **Audio**: Use a decent mic; background music optional but quiet
7. **Export**: MP4, H.264, 5-10 Mbps bitrate
